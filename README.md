# powershell-systemctl

Implements some *-Service cmdlets for Linux systems

### Functions in Systemctl.psm1

- `Get-Service`  
`sudo systemctl status $name`

- `Start-Service`  
`sudo systemctl start $name`

- `Stop-Service`  
`sudo systemctl stop $name`

- `Restart-Service`  
`sudo systemctl restart $name`
